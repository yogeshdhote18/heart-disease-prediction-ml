# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview
This project predicts the likelihood of heart disease using patient medical data. It applies machine learning techniques to identify patterns and support early diagnosis.

## 🚀 Features
- Data preprocessing and cleaning  
- Feature engineering  
- Multiple ML models (Logistic Regression, Random Forest)  
- Model evaluation using Accuracy, Precision, Recall, F1 Score  

## 📊 Results
- Achieved **85%+ accuracy**  
- Random Forest performed best among models  

## 🛠 Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  

## 📂 Project Workflow
1. Data Collection  
2. Data Preprocessing  
3. Model Training  
4. Model Evaluation  
5. Prediction  

## 📌 Conclusion
Machine learning can effectively predict heart disease risk and assist in early healthcare decisions.
